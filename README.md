## 📌 Overview

The **Motot School Information Management System (SIMS)** is a full-stack web application designed to manage academic and administrative operations for **Motot Primary and Secondary School**.

The system provides a centralized platform for managing students, teachers, courses, attendance, grades, and institutional data with **secure role-based access control**.

## 🚀 Features

### 👨‍🎓 Student Module

* Student registration with auto-generated ID
* Profile management
* View grades and GPA
* Track attendance
* Download transcripts and certificates

---

### 👨‍🏫 Instructor Module

* Manage courses
* Upload and update grades
* Record attendance
* Share learning materials

### 🏢 Admin / Registrar Module

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/kueiyiee/motot-hub.git
cd motot-hub
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Setup Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_KEY=your_supabase_key
```

---

### 4. Run Development Server

```bash
npm run dev
```
